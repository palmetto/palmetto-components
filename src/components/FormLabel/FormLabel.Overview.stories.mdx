# FormLabel

FormLabel is a sub component of many of our inputs, and is meant to be paired with a form input.

<Canvas isExpanded>
  <FormLabel>Form Label</FormLabel>
</Canvas>

### Help Text

Use `helpText` to add additional context for the FormLabel's input.
This can help keep checkbox label concise.

<Canvas>
  <Story name="Help Text">
    <FormLabel inputId="required" helpText="More helpful text about the input">
      Required input label
    </FormLabel>
  </Story>
</Canvas>

### Required

A label for an input that is required is marked with an asterisk.

<Canvas>
  <Story name="Required">
    <FormLabel inputId="required" isFieldRequired>
      Required input label
    </FormLabel>
  </Story>
</Canvas>

### Interactive Label

FormLabels support the ability to for its text to be interactive or link to another page.
