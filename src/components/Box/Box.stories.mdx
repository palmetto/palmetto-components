# Box

A `<Box>` is a layout component to build UIs with consistent padding and spacing between elements.
It lays out it contents in a specified direction, and by default renders a `div` element.

<Canvas>
  <>
    <Box
      margin="lg"
      padding="xl"
      border="grey-lighter"
      radius="md"
      fontSize="2xl"
    >
      I am a Box
    </Box>
    <Box
      margin="lg"
      padding="md"
      background="info-lighter"
      color="info"
      radius="md"
    >
      I am a Box too!
    </Box>
    <Box
      margin="lg"
      padding="md"
      background="grey-dark"
      color="inverse"
      fontSize="sm"
    >
      I am an inverse Box
    </Box>
  </>
</Canvas>

## Background Color

Use the `background` prop to set the color of the background

<Canvas>
  <Story name="background">
     {[...PALMETTO_BACKGROUND_COLOR_OPTIONS].map((color, i) => {
      if (color.includes('inherit')) return;
      else if (color === 'dark'
        || color === 'light'
        || color === 'black'
        || color === 'white'
      ) {
        return (
          <Box padding="xs" background={color} key={i}>
            {`${color} background`}
          </Box>
        );
      }
      return (
        <div key={i}>
          <Box padding="xs" background={`${color}-lightest`}>
            {`${color}-lightest background`}
          </Box>
          <Box padding="xs" background={`${color}-lighter`}>
            {`${color}-lighter background`}
          </Box>
          <Box padding="xs" background={`${color}-light`}>
            {`${color}-light background`}
          </Box>
          <Box padding="xs" background={color} key={i}>
            {`${color} background`}
          </Box>
          <Box padding="xs" background={`${color}-dark`}>
            {`${color}-dark background`}
          </Box>
          <Box padding="xs" background={`${color}-darker`}>
            {`${color}-darker background`}
          </Box>
          <Box padding="xs" background={`${color}-darkest`}>
            {`${color}-darkest background`}
          </Box>
        </div>
      )
    })}
  </Story>
</Canvas>

## Margin

The `margin` prop accepts a single [spacing](/?path=/story/design-tokens-spacing--page) value or a set of spacing values that correspond with css shorthand syntax. Margin values can be set on the horizontal or vertical axis, or the top, right, bottom and left sides of a Box.

<Canvas>
  <Story name="margin">
    {[...PALMETTO_SPACING_SIZE_OPTIONS].map((spacing, i) => (
      <Box margin={spacing} background="primary-lighter" key={i}>
        {`${spacing} margin`}
      </Box>
    ))}
  </Story>
</Canvas>

## Padding

The `padding` prop accepts a single [spacing](/?path=/story/design-tokens-spacing--page) value or a set of spacing values that correspond with css shorthand syntax. Padding values can be set on the horizontal or vertical axis, or the top, right, bottom and left sides of a Box.
