# Box

A `<Box>` is a layout component to build UIs with consistent padding and spacing between elements.
It lays out it contents in a specified direction.

## Props
<Props of={Box} />

## Font Color

Use the `color` prop to set the color of the text

<Preview>
  <Story name="Font Color">
     {[...PALMETTO_BRAND_COLOR_OPTIONS].map((color, i) => (
      <Box color={color} key={i}>
        {`${color} color`}
      </Box>
    ))}
  </Story>
</Preview>

## Font Size

Use the `fontSize` prop to set the size of the text

<Preview>
  <Story name="Font Size">
    {[...PALMETTO_FONT_SIZE_OPTIONS].reverse().map((fontSize, i) => (
      <Box fontSize={fontSize} key={i}>
        {`${fontSize} font size`}
      </Box>
    ))}
  </Story>
</Preview>
