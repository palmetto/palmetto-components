# Box

A `<Box>` is a layout component to build UIs with consistent padding and spacing between elements.

- It lays out it contents in a specified `direction`
- Renders a `<div />` element by default, but can be changed using `as`
- Allows you to specify the spacing between child elements using `childGap`

<Canvas>
  <Box childGap="md">
    <Box
      padding="xl"
      border="primary-light"
      background="primary-lightest"
      radius="md"
      fontSize="xl"
    >
      I am a box with a green background and border
    </Box>
    <Box
      childGap="sm"
      direction="row"

    </Box>

</Canvas>

## About Responsive Props
Any properties in the table above that accept a `ResponsiveProp` will take an object with the following shape:
```
{
  base?: prop, // breakpoint starting at 0 pixels, not scoped by media query.
  tablet?: prop, // 608px and higher.
  desktop?: prop, // 992px and higher.
  hd?: prop, // 1280px and higher.
}
```
These props will create classes that are scoped to the specific breakpoint they're assigned to. Not all breakpoints
are required in this shape, you can pick and choose which breakpoints require different layout attributes based on a
mobile first approach.

## Flex Properties

### Flex

The most common use case is to set children of a `<Box />` to `flex="auto"`.
The item is sized according to its width and height properties, but grows to absorb any extra free space in the parent, and shrinks to its minimum size to fit the container.
This is equivalent to setting "flex: 1 1 auto".

## Margin

The `margin` prop accepts a single [spacing](/?path=/story/design-tokens-spacing--page) value or a set of spacing values that correspond with css shorthand syntax. Margin values can be set on the horizontal or vertical axis, or the top, right, bottom and left sides of a Box.

<Canvas>
  <Story name="margin">
    <Box margin="lg md" background="primary-lighter">
      lg vertical and md horizontal
    </Box>
     <Box margin="0 2xl" background="primary-lighter">
      0 vertical and 2xl horizontal
    </Box>
    <Box margin="sm md 2xl" background="primary-lighter">
      sm top, md horizontal, 2xl bottom
    </Box>
    <Box margin="sm lg 2xl 0" background="primary-lighter">
      sm top, lg right, 2xl bottom, 0 left
    </Box>
  </Story>
</Canvas>

A single `margin` value will apply an equal amount of space around the outside of a Box.

## Responsive Styling

Props that are responsive-enabled follow a mobile-first approach, which means that styles are applied first to smaller viewports (i.e. mobile devices).
As the viewport size increases, new styles may be applied (styles are overidden for larger viewports).
Responsive enabled props take a keyed object that specifies at which [breakpoints](/?path=/docs/design-tokens-breakpoints--page) to apply certain properties.

```
<Box
  width={{
    base: '100px' // box width is 100px
    tablet: '200px' // box width is 200px when viewport width is greater than size-breakpoint-tablet
    desktop: '300px' // box width is 600px when viewport width is greater than size-breakpoint-destkop
    hd: '600px' // box width is 600px when viewport width is greater than size-breakpoint-hd
  }}
>
```
