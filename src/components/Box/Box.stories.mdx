# Box

A `<Box>` is a layout component to build UIs with consistent padding and spacing between elements.
It lays out it contents in a specified direction, and by default renders a `div` element.

## Props
<Props of={Box} />

## Background Color

Use the `background` prop to set the color of the background

<Preview>
  <Story name="background">
     {[...PALMETTO_BACKGROUND_COLOR_OPTIONS].map((color, i) => {
      if (color.includes('inherit')) return;
      else if (color === 'dark'
        || color === 'light'
        || color === 'black'
        || color === 'white'
      ) {
        return (
          <Box padding="xs" background={color} key={i}>
            {`${color} background`}
          </Box>
        );
      }
      return (
        <>
          <Box padding="xs" background={`${color}-lighter`} key={i}>
            {`${color}-lighter background`}
          </Box>
          <Box padding="xs" background={`${color}-light`} key={i}>
            {`${color}-light background`}
          </Box>
          <Box padding="xs" background={color} key={i}>
            {`${color} background`}
          </Box>
          <Box padding="xs" background={`${color}-dark`} key={i}>
            {`${color}-dark background`}
          </Box>
          <Box padding="xs" background={`${color}-darker`} key={i}>
            {`${color}-darker background`}
          </Box>
        </>
      )
    })}
  </Story>
</Preview>
