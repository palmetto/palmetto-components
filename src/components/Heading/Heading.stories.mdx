# Heading

Headings are used as titles for a page or section of a page.

<Preview isExpanded>
  <Heading as="h1">
    Lead the world towards a clean energy future
  </Heading>
</Preview>

## Props

<Props of={Heading} />

## All Levels

Each Heading level has a default size.

  <Story name="Levels">

## Sizes

The default size of the Heading can overridden as xs-5xl so that it's appropriate for its surrounding context.

<Preview>
  <Story name="Sizes">
    <>
      {[...PALMETTO_FONT_SIZE_OPTIONS].reverse().map(fontSize => (
        <Heading size={fontSize} key={uuid()}>
          {`Size ${fontSize}`}
        </Heading>
      ))}
    </>
  </Story>
</Preview>

## Colors

The color of the heading is inherited from its parent, but can be set to brand color as well.

<Preview>
  <Story name="Colors">
    <>
      {PALMETTO_FONT_COLOR_OPTIONS.map(color => (
        <Heading color={color} key={uuid()}>
          {color}
        </Heading>
      ))}
    </>
  </Story>
</Preview>
