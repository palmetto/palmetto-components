# Modal

Modals are containers appearing in front of the main content to provide critical information or an actionable piece of content.

<Canvas>
  <Story name="Example">
    {() => {
      const [showDialog, setShowDialog] = React.useState(false);
      const open = () => setShowDialog(true);
      const close = () => setShowDialog(false);
      return (
        <div>
          <button onClick={open}>Show Modal</button>
          <Modal isOpen={showDialog} onDismiss={close}>
            <p>The overlay styles are a white fade instead of the default black fade.</p>
            <button onClick={close}>Very nice.</button>
          </Modal>
        </div>
      );
    }}
