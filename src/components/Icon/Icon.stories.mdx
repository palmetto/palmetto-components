# Sizing

The size of the icon is inherited from the font-size. This allows for easy inline placement inside of other components, text, etc.
If you wish to size your icon independent of its container, you can pass it a utility class, (E.G: `font-size-md`) or declare the font
size as a style object.

<Canvas>
  <Story name="Sizing">
    <Box childGap="md" alignItems="flex-start">
      <Button><Icon name="user" />&nbsp;&nbsp;Inside a button</Button>
      <Heading size="xl" as="h2"><Icon name="user" />&nbsp;&nbsp;In a heading</Heading>
    </Box>
  </Story>
</Canvas>

# Fallback

If the icon name is not found in the dictionary, we'll display `N/A` as a fallback, as well as log an error to the console.
<Canvas>

          borderColor="grey-lighter"
          padding="md"
          radius="md"

            <Icon name="does-not-exist" className="font-size-2xl"/>
          </Heading>

      );
