# Sizing

The size of the icon is inherited from the font-size. This allows for easy inline placement inside of other components, text, etc.
If you wish to size your icon independent of its container, you can pass it a utility class, (E.G: `font-size-md`) or declare the font
size as a style object.

<Canvas>
  <Story name="Sizing">

      <Heading size="xl" as="h2"><Icon name="user" />&nbsp;&nbsp;In a heading</Heading>
    </Box>
  </Story>
</Canvas>
