## Sizing

The size of the icon is inherited from its parent's font-size. This allows for easy inline placement inside of other components, text, etc.
If you wish to size your icon independent of its container, you can pass it a utility class, (E.G: `font-size-md`) or declare the font
size as a style object.

<Canvas>
  <Story name="Sizing">
    <Box as="p" alignItems="center" direction="row" childGap="xs" fontSize="xl">
      <Icon name="user" /> font size xl
    </Box>
  </Story>
</Canvas>

## Color

Similarly to sizing, icons inherit their color from the font-color.

## Fallback

If the icon name is not found in the dictionary, we'll display `N/A` as a fallback, as well as log an error to the console.
