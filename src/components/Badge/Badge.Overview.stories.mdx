## Variants

Badges come in different variants based on the kind of message being delivered to the user.

<Canvas>
  <Story name="Variants">
    {() => {
      const variants = [
        'default',
        'primary',
        'secondary',
        'tertiary',
        'info',
        'success',
        'warning',
        'danger',
      ];
      return (
        <Box direction="row" childGap="sm">
          {variants.map(variant => (
            <Badge message={variant} variant={variant} key={variant} />
          ))}
        </Box>
      );
    }}
  </Story>
</Canvas>

## Sizes

Set the size of the badge to `sm`, `md`, `lg` or `xl`. `md` is the default size.

<Canvas>
  <Story name="Sizes">
    {() => {
      return (
        <>
          <Box direction="column" alignItems="flex-start" childGap="md">
            <Badge size="sm" message="Small" />
            <Badge size="md" message="Medium" />
            <Badge size="lg" message="Large" />
            <Badge size="xl" message="X-Large" />
          </Box>
        </>
      );
    }}
  </Story>
</Canvas>

## Custom Class

You can apply a custom class to your badge with the `className` prop. Here we have added a 3xl margin.

<Canvas>
  <Story name="Custom Class">
    {() => {
      return (
        <Box direction="row">
          <Badge message="Hello world!" variant="success" className="m-3xl" />
        </Box>
      );
    }}
  </Story>
</Canvas>
