## Variants

Badges come in different variants based on the kind of message being delivered to the user.

<Canvas>
  <Story name="Variants">
    {() => {
      const variants = [
        'default',
        'primary',
        'secondary',
        'tertiary',
        'info',
        'success',
        'warning',
        'danger',
      ];
      return (
        <Box direction="row" childGap="sm">
          {variants.map(variant => (
            <Badge message={variant} variant={variant} key={variant} />
          ))}
        </Box>
      );
    }}
  </Story>
</Canvas>

## Sizes

Set the size of the badge to `sm`, `md`, `lg` or `xl`. `md` is the default size. A `ResponsiveProp` object may also be used
so the badge can be a different size per breakpoint.

<Canvas>
  <Story name="Sizes">
    {() => {
      return (
        <>
          <Box direction="column" alignItems="flex-start" childGap="md">
            <Badge size="sm" message="Small" />
            <Badge size="md" message="Medium" />
            <Badge size="lg" message="Large" />
            <Badge size="xl" message="X-Large" />
            <Badge
              size={{
                base: 'sm',
                tablet: 'md',
                desktop: 'lg',
                hd: 'xl',
              }}
              message="Responsive"
            />
          </Box>
        </>
      );
    }}
  </Story>
</Canvas>
